# java-tutorial

Java の基礎（コンソールサンプル）から、同じ題材の Spring Boot アプリまでを一つのリポジトリで辿れる教材用プロジェクトです。

## どこから読むか

| 内容 | 場所 |
|------|------|
| レッスン一覧・コンパイル・実行 | [JavaSample/README.md](JavaSample/README.md) |
| Spring Boot デモ・ガイド・アーキテクチャ | [JavaApp/README.md](JavaApp/README.md) |

## ざっくり構成

- **JavaSample** — 文法から OOP までのレッスン（`lesson01` 〜 `lesson21`）
- **JavaApp/demo** — 図書貸出を題材にした Spring Boot（`JavaSample` の lesson21 と対応）

詳しいディレクトリ構成とレイヤー説明は [JavaApp/docs/ARCHITECTURE.md](JavaApp/docs/ARCHITECTURE.md) を参照してください。
