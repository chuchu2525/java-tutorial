# Lesson03 Loops

対応コード: `lesson03_loops/Lesson03Loops.java`

## このファイルで学ぶこと

- `for` 文
- `while` 文
- `for` と `while` の使い分け
- 繰り返し処理の基本

## コードの概要

このサンプルでは、`for` 文と `while` 文を使って、次の違いを体験できます。

- `for` 文: 回数が決まっている繰り返し
- `while` 文: 条件を満たす間、繰り返し続ける処理

## 注目ポイント

### 1. `for` 文は「回数ベース」の繰り返しに向いている

```java
for (int i = 1; i <= 5; i++) {
    System.out.println("for -> " + i);
}
```

`for` は初期化・条件・更新が1か所にまとまるため、  
「何回まわすか」が読み取りやすいです。

### 2. `while` 文は「条件ベース」の繰り返しに向いている

```java
int sum = 0;
int n = 1;
while (sum <= 20) {
    sum += n;
    System.out.println("while -> n=" + n + ", sum=" + sum);
    n++;
}
```

この例では「合計 `sum` が 20 以下の間だけ続ける」という条件です。  
何回で終わるかを先に決めずに書けるのが `while` の強みです。

> 注意: ループの中で条件に関わる値（この例では `sum` や `n`）を更新しないと、  
> 無限ループになることがあります。

## 実行するとどうなるか

1. `for` のセクションで `1` から `5` まで順番に表示される  
2. `while` のセクションで `sum` が増えていき、`20` を超えたら終了する
