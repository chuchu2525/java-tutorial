# Lesson03 Loops

対応コード: `lesson03_loops/Lesson03Loops.java`

## このファイルで学ぶこと

- `for` 文
- `while` 文
- 繰り返し処理の基本
- カウンタ変数の使い方

## コードの概要

このサンプルでは、同じ処理を複数回実行する方法として、`for` 文と `while` 文を両方使っています。

どちらも「条件を満たす間、繰り返す」ための文ですが、書き方が少し違います。

## 注目ポイント

### 1. `for` 文は回数が決まっているときに便利

```java
for (int i = 1; i <= 3; i++) {
    System.out.println(i + "回目");
}
```

この1行に、初期化・条件・更新がまとまっています。

### 2. `while` 文は条件を見ながら繰り返す

```java
int count = 1;
while (count <= 3) {
    System.out.println(count + "回目");
    count++;
}
```

`count++` を忘れると、条件がずっと真のままになって無限ループになることがあります。

## 実行するとどうなるか

最初に `for文` と表示されて3回出力され、その後 `while文` と表示されて同じように3回出力されます。

## 自分で試すなら

- 回数を `5` 回に変える
- `i = 0` から始めて違いを見る
- `2` ずつ増えるように `i++` を `i += 2` に変える
- `while` 文で `count` を `10` から減らしていく形に変える
