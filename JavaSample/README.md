# JavaSample

Java を順番に学べるように、レッスンごとにディレクトリを分けたサンプル集です。

## 学習の進め方

1. 各 lesson ディレクトリの `.md` を読む
2. 対応する `.java` を読む
3. コンパイルして実行する
4. `自分で試すなら` の項目を実際に変更してみる
