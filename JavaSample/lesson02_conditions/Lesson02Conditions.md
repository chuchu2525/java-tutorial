# Lesson02 Conditions

対応コード: `lesson02_conditions/Lesson02Conditions.java`

## このファイルで学ぶこと

- `if`
- `else if`
- `else`
- 三項演算子

## コードの概要

このサンプルでは、`score` の値によって表示するメッセージを変えています。

最初は `if` 文で細かく分岐し、後半では三項演算子を使って、より短く条件分岐を書いています。

## 注目ポイント

### 1. 条件が真ならそのブロックが実行される

```java
if (score >= 80) {
    System.out.println("良い点です");
}
```

`score >= 80` が真のときだけ中身が実行されます。

### 2. 複数条件は `else if` でつなぐ

```java
if (score >= 80) {
    ...
} else if (score >= 60) {
    ...
} else {
    ...
}
```

上から順に評価され、最初に当てはまった処理だけが動きます。

### 3. 短い条件分岐は三項演算子でも書ける

```java
String message = (score >= 60) ? "合格" : "不合格";
```

これは「条件が真なら左、偽なら右」を返す式です。

## 実行するとどうなるか

`score` が `75` なので、`if` 文では「合格です」が表示され、三項演算子では「判定: 合格」が表示されます。

- `score` を `50` にして結果を見る
- 合格ラインを `70` に変えてみる
- `score` を2つ用意して、より高い方だけ判定する
